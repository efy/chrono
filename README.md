
# Chrono

Tool for extracting time from git log.

# Installation

Grab the latest binary from [releases](https://github.com/efy/chrono/releases) page for your architecture and place it 
somewhere on your `PATH`.

# Usage

```sh
chrono -author={name}
=> 16 hours
```
